# Xideral Academy - Ejercicios Prácticos Semana 3

### Edson Miguel Peralta Valdez

Este un proyecto java el cual contiene los ejercicios prácticos realizados en la semana 3 de la Academia Xideral FullStack

## Temas vistos
* Colecciones avanzadas: List, Set, Map, Queue
* Generics (clases, interfaces, métodos)
* Comparator vs Comparable
* Lambdas y functional interfaces
* Optional
* Streams: creación, operaciones terminales e intermedias
* Primitive streams, pipelines avanzados
* Threads, concurrencia, ExecutorService

## Pasos para correr el proyecto

1. Clonar el repositorio
2. Entrar a cada paquete con su respectivo ejercicio y ejecutar el metodo main
